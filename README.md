# dmx2artnet
Converts serial DMX to UTP ArtNet using a Beck IPC board. Watcom development suite required.<br />
Build:<br />

setup correct environment before compiling.
[see here](https://tkkrlab.nl/wiki/Aan_de_slag_met_Beck-board)

```
wmake dmxin.exe   
wmake dmxout.exe  
```

Clean
```
wmake clean
```
